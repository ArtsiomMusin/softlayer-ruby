    class SoftLayer::ObjectFilterBlockHandler

    This class defines the routines that are valid within the block provided to a call to SoftLayer::ObjectFilter.build. This allows you to create object filters like:

    object_filter = SoftLayer::ObjectFilter.build("hardware.memory") { is_greater_than(2) }
